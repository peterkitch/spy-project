## Overview

ImpactSearch analyzes statistical relationships between different tickers, helping identify:
- Correlation patterns
- Lead-lag relationships
- Market impact analysis
- Cross-ticker dependencies

## Key Features

- Multi-ticker statistical analysis
- Configurable analysis windows
- Excel export functionality
- Visual correlation matrices
